About

This website was made to expose individuals to the history of major operating systems which have influenced our daily lives with the help of web graphics. This webpage, WinPedia, was originally meant to be its own website. However, imagination has transformed this into a larger project, OSHistory. It is now a very large project in development since December of 2024. Some graphics are created by me due to lack of pictures on the internet, which notably include the given backgrounds for Windows NT 4.0, Windows 2000 and Windows Me, as well as the logos relating to this website such as the OSHistory logo and WinPedia logo. There are no credits for the pictures as I forgot to document where the images came from. However, all Windows logos are created and owned by Microsoft and are used here for educational purposes.
